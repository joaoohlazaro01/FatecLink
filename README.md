# FatecLink

## 📌 Sobre o projeto

O **FatecLink** é uma plataforma web desenvolvida para conectar alunos da Fatec a empresas que oferecem oportunidades de estágio e emprego.

O sistema foi criado com o objetivo de centralizar a divulgação de vagas, facilitando o processo de recrutamento para as empresas e tornando a busca por oportunidades mais simples e organizada para os estudantes.

---

## 🎯 Objetivo

Criar uma plataforma que permita:

* Cadastro de empresas;
* Cadastro de alunos;
* Publicação de vagas;
* Busca de oportunidades;
* Candidatura às vagas;
* Administração de usuários e conteúdos.

---

## 🚀 Funcionalidades

### 👨‍🎓 Área do Aluno

* Cadastro e login
* Edição do perfil
* Upload de currículo
* Busca de vagas
* Filtros por categoria
* Visualização de detalhes das vagas
* Candidatura às oportunidades
* Histórico de candidaturas

### 🏢 Área da Empresa

* Cadastro e login
* Gerenciamento do perfil da empresa
* Publicação de vagas
* Edição e exclusão de vagas
* Visualização dos candidatos
* Gerenciamento das oportunidades

### 👨‍💼 Área Administrativa

* Login administrativo
* Gerenciamento de usuários
* Gerenciamento de empresas
* Gerenciamento de vagas
* Controle do sistema
* Relatórios
